ðŸ”§ How It Works

Each token uses the same high-quality Lissajous curve generation with 720-point oversampling, smoothing, and quadratic path compression. The block hash determines the color palette (1 of 20), while additional block characteristics create unique hue shifts, saturation boosts, and glow intensities.

Rare variants (5.9%) add silver tints, Epic variants (0.37%) add golden shimmer, and Legendary variants (0.02%) feature rainbow effects and special backgrounds.

Every single token is deterministically generated from its block data, ensuring true uniqueness while maintaining consistent visual quality across all 10,000 pieces.
